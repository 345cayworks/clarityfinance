# Bank Partner Workflow

Clarity Finance is a customer preparation and readiness platform, not an underwriting engine.

## Bank-Friendly Workflow

1. User completes a financial snapshot.
2. User reviews cash flow, debt pressure, and readiness indicators.
3. User runs planning scenarios.
4. User prepares a document checklist.
5. User gives explicit consent to share a summary.
6. Advisor or bank officer receives a readiness packet.
7. Bank performs final underwriting outside Clarity Finance.

## Compliance-Safe Phrases

- Readiness estimate.
- Readiness range.
- Document readiness.
- Applicant preparation.
- Lender conversation prep.
- Final approval subject to lender underwriting.
- Advisor review recommended.
- Lender review required.

## Phrases to Avoid

- Guaranteed approval.
- User qualifies.
- Bank-approved.
- Underwriting decision.
- Approved amount.
- Best investment.
- Guaranteed return.

## Positioning

Clarity Finance helps users organize and understand financial information before speaking with licensed advisors, lenders, or financial institutions. It can improve preparedness, document completeness, and clarity of lender conversations, but all credit decisions remain with the financial institution.
